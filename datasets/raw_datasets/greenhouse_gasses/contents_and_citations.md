# What's here

Atmospheric concentration of the four main greenhouse gasses

- Carbon Dioxide (CO<sub>2</sub>) in parts per million (ppm)
- Methane (CH<sub>4</sub>) in parts per billion (ppb)
- Nitrous Oxide (N<sub>2</sub>O) in parts per billion (ppb)
- Sulfur Hexaflouride (SF<sub>6</sub>) in parts per trillion (ppt)

## Entry format
- Dataset
- Any time frame sub-datasets
- File path(s) from the root of this project
    - File name format: `<source>-<file_contents>-<date: YYYYMMDD>`
- Source citations
