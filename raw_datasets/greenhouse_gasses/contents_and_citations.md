# What's here

Atmospheric concentration of the four main greenhouse gasses

- Carbon Dioxide (CO~2~) in parts per million (ppm)
- Methane (CH~4~) in parts per billion (ppb)
- Nitrous Oxide (N~2~O) in parts per billion (ppb)
- Sulfur Hexaflouride (SF~6~) in parts per trillion (ppt)

## Entry format
- Dataset
- Any time frame sub-datasets
- File path(s) from the root of this project
    - File name format: `<source>-<file_contents>-<date: YYYYMMDD>`
- Source citations
