# What's here
- Temperature average
- Anomaly: yearly average temperature difference from a baseline mean temperature. See the[NOAA climate-at-a-glance](https://www.ncei.noaa.gov/access/monitoring/climate-at-a-glance/global/data-info) for more information.

## Entry format
- Dataset
- Any regional sub-datasets
- File path(s) from the root of this project
    - File name format: `<source>-<file_contents>-<date: YYYYMMDD>`
- Source citations
